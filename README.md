# coworker-bot

Listens for events on GitHub, Linear, and Slack, then dispatches a [Crafting Coding Agent](https://www.crafting.dev/) to handle them — running continuously inside a Crafting Sandbox.

## Quick Start

**[→ GitHub Quick Start](docs/quickstart.md)** — get up and running in ~10 minutes.

For multi-provider setups, security hardening, and advanced configuration, see the **[full setup guide](docs/setup/README.md)**.
